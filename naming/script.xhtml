Script for natural language

Rule

A single native script MUST be used whenever possible. If it is not possible, standard romanisation MUST be used.

Explanation

Whenever the use of native script of a natural language is possible, it must be used. Only when it can't be used can full romanisation be used.

Examples

Chinese characters can be used directly in PHP identifiers. Therefore, when making a PHP project in Chinese, Chinese characters must be used in the names.

For Mandarin Chinese, in case Chinese characters can't be used, only full pinyin may be used. For example, 中文 must be written as Zhongwen, but not any other spellings such as zh or zw.

For Mongolian, if the coding environment does not support traditional Mongolian script, Cyrillic script must be used if possible. However, if traditional Mongolian script is used, mixing in Cyrillic is not allowed.
