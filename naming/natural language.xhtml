Natural language

Rule

All names in a single project MUST be in a single language which is used by the target audience.

Explanation

A project must be written in a single natural language. Mixing different natural languages in names is not allowed.

Examples

A project developed solely for use in Hong Kong must be in either traditional Chinese or English. Although the interface should support both languages, the names in the code must be in either one but not both.
