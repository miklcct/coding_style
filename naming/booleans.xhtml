Booleans

Rule

Boolean non-function names MUST be an adjective. Boolean function verbs MUST be in third-person form.

Explanation

All boolean non-functions must be an adjective, specifying if the adjective applies.

All functions returning a boolean must be in third-person form, specifying if the verb applies.

Examples

Variables

  • safe
  • full
  • havingChildren
  • acceptingQueries

Functions

  • isSafe
  • isFull
  • hasChildren
  • acceptsQueries
