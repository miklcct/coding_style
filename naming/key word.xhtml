Key word

Rule

All key words MUST be in lower case in non-case sensitive languages.

Explanation

All key words must be in lower case in non-case sensitive languages, for example, in PHP, SQL, or Windows batch scripts.

Language-specific notes

PHP

The words TRUE, FALSE and NULL are not key words. They are built-in constants and must be typed in upper case.

The word class in the expression ::class used to get a fully-qualified class name is a key word, therefore it must be typed in lower case. It is not a constant as it is not listed in ReflectionClass::getConstants.
