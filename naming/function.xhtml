Function

Rule

Function names MUST start with a verb.

Explanation

All functions must start with a verb.

Examples

Non-member functions

  • get_request
  • say_sorry
  • crash

Member functions

  • fire
  • submit
  • increaseGear
  • isSpeeding

Language-specific notes

Javascript

If the function is intended to be used as a constructor, this rule does not apply.
